# Naive Bayes Sentiment Classifier  

## Description  
This project implements a Naive Bayes classifier to classify movie reviews as positive or negative. The system preprocesses textual data, extracts features using TF-IDF, and evaluates performance on a test dataset with metrics like accuracy, precision, recall, and F1-score.

---

## Features  
1. **Document Preprocessing:**  
   - Tokenizes reviews, removes punctuation and stopwords, and applies stemming.  

2. **Naive Bayes Classification:**  
   - Trains a Multinomial Naive Bayes classifier on TF-IDF features (top 5,000 terms).  

3. **Model Evaluation:**  
   - Evaluates performance with accuracy, precision, recall, F1-score, and a detailed classification report.
